## 🚀 Quick Start

1. **Clona el repositorio**

   ```bash
   git clone https://github.com/gortizmartija/Porfolio.git
   ```

2. **Instala dependencias**

   ```bash
   pnpm install
   ```

3. **Inicia el servidor de desarrollo**
   ```bash
   pnpm dev
   ```
